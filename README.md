Portfolio Website – Sumit Soni <br><br>
Overview<br>
<br>
This project is a simple, responsive personal portfolio website built using HTML, CSS, JavaScript, Font Awesome icons, and the AOS (Animate On Scroll) library. It presents a clean single-page layout to showcase the developer’s identity, skills, social profiles, and contact information.

Features
Fixed navigation bar with links for Home, Skills, Introduction, Services, and Contact sections.

Animated logo and navigation items using AOS scroll animations.

Hero section with a heading, typewriter-style role text (Web Designer), and a short professional introduction paragraph.

Social media icons (LinkedIn, GitHub, X/Twitter) linked to actual profiles using Font Awesome.

Contact information including phone number and email.

Profile image displayed alongside the introduction.

Mobile-friendly navigation using a hamburger menu and dropdown (handled via script.js).

Technologies Used
HTML5 for structure

CSS3 (style.css) for styling and layout

JavaScript (script.js) for hamburger menu toggle logic

Font Awesome 6.5.1 for social media and menu icons

AOS (Animate On Scroll) for scroll-based animations
